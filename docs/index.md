Bicep starter modules for SEPPmail-adjacent Azure infrastructure and operational guardrails.

Use these modules to bootstrap modern Azure landing-zone aligned deployments with diagnostics and policy hooks.

- Security, compliance and control mapping content
- Example scripts and reusable patterns
- Operations, troubleshooting and observability guidance
- Demo scenarios, release notes and community entry points
