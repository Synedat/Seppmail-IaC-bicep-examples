# Seppmail-IaC-bicep-examples

> Bicep starter examples for Azure-based SEPPmail infrastructure patterns.

**Partner resource by Synedat for the SEPPmail ecosystem.**

## Why this repository exists

Azure Bicep examples for resource groups, networking and gateway-adjacent infrastructure that can support SEPPmail deployments.

This repository is structured for public consumption and easy discovery across topics such as SEPPmail, Exchange Online, Microsoft 365, Azure, API automation, PowerShell and operational runbooks.

## Included content

- `main.bicep`
- `parameters.example.json`
- `modules/network.bicep`

## Quick start

1. Use these Bicep files as a scaffolding layer for internal Azure standards.
2. Integrate organization-specific NSGs, diagnostics, policy and naming conventions.
3. Treat the examples as composable building blocks, not finished production blueprints.
